HTML5

CSS3

Styling language used for describing the presentation of documents written in HTML.

JavaScript

High-level, interpreted programming language that conforms to the ECMAScript specification.

React

JavaScript library for building user interfaces, maintained by Facebook and a community of developers.

Vue.js

Progressive JavaScript framework for building user interfaces with an approachable core library.

Bootstrap

Free and open-source CSS framework directed at responsive, mobile-first front-end web development.

Sass

Preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets.

Angular

TypeScript-based free and open-source web application framework led by the Angular Team at Google.

Node.js

Open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine.

Python

High-level, interpreted, general-purpose programming language with dynamic semantics.

PHP

Server-side scripting language designed for web development but also used as a general-purpose programming language.

Laravel

Free, open-source PHP web framework intended for the development of web applications following MVC architectural pattern.

Java

Class-based, object-oriented programming language designed to have as few implementation dependencies as possible.

Express.js

Minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

Ruby on Rails

Server-side web application framework written in Ruby under the MIT License that provides default structures for a database.

ASP.NET

Developer platform made up of tools, programming languages, and libraries for building many different types of applications.

Amazon Web Services

Comprehensive, evolving cloud computing platform provided by Amazon that includes a mixture of infrastructure-as-a-service.

Google Cloud

Suite of cloud computing services that runs on the same infrastructure that Google uses internally for its end-user products.

Microsoft Azure

Cloud computing service created by Microsoft for building, testing, deploying, and managing applications and services.

Docker

Set of platform as a service products that use OS-level virtualization to deliver software in packages called containers.

Kubernetes

Open-source container-orchestration system for automating computer application deployment, scaling, and management.

Jenkins

Free and open-source automation server that helps automate the parts of software development related to building, testing, and deploying.

Terraform

Infrastructure as Code software tool that allows you to safely and predictably create, change, and improve infrastructure.

Ansible

Open-source software provisioning, configuration management, and application-deployment tool enabling infrastructure as code.

MySQL

Open-source relational database management system based on SQL - Structured Query Language.

PostgreSQL

Powerful, open source object-relational database system with over 30 years of active development.

MongoDB

Source-available cross-platform document-oriented database program classified as a NoSQL database program.

Redis

In-memory data structure store, used as a distributed, in-memory key–value database, cache and message broker.

Firebase

Platform developed by Google for creating mobile and web applications with tools and infrastructure.

MariaDB

Community-developed, commercially supported fork of the MySQL relational database management system.

Elasticsearch

Search engine based on Lucene library that provides a distributed, multitenant-capable full-text search engine.

SQLite

C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine.
